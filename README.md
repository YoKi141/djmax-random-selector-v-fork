# DJMAX Random Selector V

---

## 简体中文

### 简介

DJMAX RESPECT V 随机曲目选择器。设定筛选条件后，按下快捷键即可自动随机选曲，无需手动翻找。

### 安装

1. 在 [Releases](https://github.com/wowvv0w/djmax-random-selector-v/releases) 页面下载最新版本。
2. 解压文件。
3. 运行 `DJMAX Random Selector V.exe`。

### 初次设置

1. 点击右下角设置按钮。
2. 在曲目列表设置（Track List Setting）中勾选您拥有的 DLC，点击 APPLY。
3. 根据需要配置筛选条件。
4. 在 DJMAX RESPECT V 自由模式（Freestyle）中，设置：
   - 分类：全部曲目（All Track）
   - 排序：按标题升序（Sort by Title A to Z）
5. 若游戏语言**不是韩语**，请在选择器设置中启用 **GAME LANGUAGE (ENGLISH)**。

### 使用方法

- 按 `F7`：随机选曲并自动定位
- 按 `Alt + F7`：重复上次选曲（不重新随机）

## 日本語

### 概要

DJMAX RESPECT V 用のランダム選曲ツールです。フィルターを設定してホットキーを押すだけで、自動的にランダムな曲を選択・ナビゲートします。

### インストール

1. [Releases](https://github.com/wowvv0w/djmax-random-selector-v/releases) ページから最新版をダウンロードします。
2. ファイルを解凍します。
3. `DJMAX Random Selector V.exe` を実行します。

### 初期設定

1. 右下の設定ボタンをクリックします。
2. 「Track List Setting」で所有している DLC にチェックを入れ、APPLY をクリックします。
3. 希望のフィルター条件を設定します。
4. DJMAX RESPECT V のフリースタイルモードで以下を設定します：
   - カテゴリ：All Track
   - ソート：タイトル昇順（Sort by Title A to Z）
5. ゲーム言語が**韓国語以外**の場合、設定で **GAME LANGUAGE (ENGLISH)** を有効にしてください。

### 使い方

- `F7`：ランダムに曲を選択・移動
- `Alt + F7`：前回の選曲を再実行（再ランダムなし）

## English

### Introduction

A random music selector for DJMAX RESPECT V. Set your filter criteria, press the hotkey, and the program automatically picks a random track and navigates to it in-game.

### Installation

1. Download the latest release from the [Releases](https://github.com/wowvv0w/djmax-random-selector-v/releases) page.
2. Extract the files.
3. Run `DJMAX Random Selector V.exe`.

### Initial Setup

1. Click the settings button at the bottom right.
2. In **Track List Setting**, select the DLCs you own and click **APPLY**.
3. Configure the filter options as desired.
4. In DJMAX RESPECT V Freestyle Mode, set:
   - Category: **All Track**
   - Sort: **Sort by Title (A to Z)**
5. If your game language is **not Korean**, enable **GAME LANGUAGE (ENGLISH)** in the selector's settings.

### Usage

- `F7` — Randomly select a track and auto-navigate to it
- `Alt + F7` — Repeat the last selection without re-rolling
